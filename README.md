 RPG-игра 

Консольная RPG с системой экипировки, инвентаря, уровней и характеристик.

На даннный момент игра не закончена.

  Возможности
- Создание героя
- Инвентарь с категориями предметов
- Экипировка брони и оружия (слоты: шлем, нагрудник, поножи, ботинки, оружие)
- Расчет характеристик (сила, ловкость, интеллект → здоровье, мана, выносливость)
- Система уровней и опыта
- Снятие и надевание предметов

  Установка и запуск
1. Склонировать репозиторий
2. Запустить файл `main.py` (или как у тебя называется)

   Пример использования
 python
 Создание предметов
crown = Helmet('Шлем Господства', 'Шлем', 'Сила', 5, 'Ловкость', 7, 'Интеллект', 3)

 Создание героя
hero = Hero('Артас')

 Экипировка
hero.equip_armor(inventory, 'Шлем', 'Шлем Господства')

 Информация о герое
hero.info_characteristic()
